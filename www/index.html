This plugin is a thin wrapper for Google Maps Android API and Google Maps SDK for iOS. Both PhoneGap and Apache Cordova are supported.

The benefit of this plugin is you can create your application using HTML, CSS and JavaScript you are familiar with, but Google Map is native.
The Google Maps native API is able to display 3D map view, faster rendering and reduce network communications (which means save battery life).
You don't need to learn native languages, such as Java or Objective-C. This plugin helps you to control the native Google Maps through JavaScript.
